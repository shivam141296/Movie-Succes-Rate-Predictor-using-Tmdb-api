# Moviesapi
Display movies in card and recycler view from tmdb website.  Features-- 1) shows movie trailers 2) shows all details of movie and their images using glide library 3) Display Top comments and rate movie using reviews from youtube by sentimental analysis in python 4)You can also create your own movie by adding your own actore,buget and run time and it will display what will be rating of movie
